### 222_Goman_WPF_PROJ
Студент: Гоман Антон Олегович

Группа: 4ИСИП-222

#### Описание проекта
WPF-приложение для учёта платежей пользователей. Реализованы:
- Авторизация и регистрация с хешированием паролей
- Капча после 3 ошибок
- Работа с базой данных SQL Server
- CRUD-операции для таблиц
- Диаграммы платежей
- Экспорт данных в Word и Excel
- Темы оформления (светлая/тёмная)
#### Технологии
- C#, WPF (.NET Framework)
- Microsoft SQL Server
