### Описание проекта
Данное приложение, реализованное на платформе WPF, предназначено для ведения учета пользовательских платежей. 
В его функционал входят:
- Авторизация и регистрация пользователей с использованием хеширования паролей;
- Отображение капчи после трех неудачных попыток входа;
- Интеграция с базой данных SQL Server;
- Выполнение операций добавления, изменения, удаления и просмотра данных (CRUD) для различных таблиц;
- В системе представлены диаграммы, демонстрирующие статистику платежей;
- Предусмотрена возможность экспорта данных в форматы Word и Excel;
- Пользователь может переключаться между светлой и тёмной темами оформления.

---
#### Используемые технологии
- Язык программирования C#, фреймворк WPF (.NET Framework)
- Система управления базами данных Microsoft SQL Server
